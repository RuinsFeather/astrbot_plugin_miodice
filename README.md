# astrbot_plugin_miodice

一款适用于 AstrBot 的基础 TRPG 骰子插件，面向跑团中的检定、对抗、伤害与角色状态管理。

## 功能概览

- 支持普通骰点与 D100 技能检定
- 支持双方对抗判定
- 支持伤害结算、护甲减伤与 HP 状态判定
- 支持 KP 暗骰
- 支持按会话保存的角色状态查看与修改
- 内置 `.help` 帮助系统，可按指令查看详细用法

## 指令一览

| 指令 | 说明 | 示例 |
| --- | --- | --- |
| `.r` / `.roll` / `.投掷` / `.检定` | 投掷骰子；当使用 `1d100` 并附带技能值时，会自动进行技能检定 | `.r 1d100 侦查60` |
| `.vs` / `.战斗` | 双方 D100 对抗检定；同级成功时，原始点数更高者胜 | `.vs 攻击55 闪避40` |
| `.dmg` / `.伤害` | 计算武器伤害、伤害加成、护甲、当前 HP 与最大 HP | `.dmg 1d10+2 伤害加成1 护甲2 hp12 hp8` |
| `.rh` / `.暗投` | KP 暗骰；群聊仅提示，私聊可直接查看结果 | `.rh 1d100` |
| `.st` / `.状态` | 查看或更新角色状态 | `.st 林默 HP 10 中毒` |
| `.help` / `.帮助` | 查看指令总览或单条指令的详细说明 | `.help dmg` |

## 详细用法

### 1. 骰点与检定

```text
.r [骰子表达式] [技能名称+技能值]
```

示例：

```text
.r 1d100 侦查60
.r 2d6+3
```

规则说明：

- `01`：大成功
- `99+`：大失败
- `<= 技能值`：成功
- `<= 技能值 / 2`：困难成功
- `<= 技能值 / 5`：极难成功

### 2. 对抗检定

```text
.vs [主动方技能名+技能值] [被动方技能名+技能值]
```

示例：

```text
.vs 攻击55 闪避40
```

判定规则：

- 成功等级更高的一方获胜
- 若成功等级相同，则原始点数更高的一方获胜
- 若两边结果完全相同，则判定为平局

### 3. 伤害结算

```text
.dmg [伤害骰] 伤害加成[数值] 护甲[数值] hp[最大HP] hp[当前HP]
```

示例：

```text
.dmg 1d10+2 伤害加成1 护甲2 hp12 hp8
```

规则说明：

- 最终伤害 = 武器骰 + 伤害加成 - 护甲
- 当单次伤害达到最大 HP 的一半时，触发重伤阈值
- 结算后若 HP `<= 0`，进入濒死
- 结算后若 HP `<= -10`，判定死亡

### 4. KP 暗骰

```text
.rh [骰子表达式]
```

示例：

```text
.rh 1d100
```

说明：

- 群聊中会尽量私发给发起者
- 群内只会显示“KP进行了暗骰”之类的提示

### 5. 角色状态

```text
.st [角色名]
.st [角色名] HP [数值] MP [数值] [状态]
```

示例：

```text
.st 林默
.st 林默 HP 10 中毒
.st 林默 -中毒
```

说明：

- 状态按会话保存
- `-状态名` 可移除对应状态
- `HP` 和 `MP` 可单独更新

## 帮助系统

- 输入 `.help` 查看全部指令总览
- 输入 `.help dmg`、`.help vs` 之类的命令查看单条指令说明
- 也可以使用别名，例如 `.help 伤害`、`.help 战斗`

## 安装与使用

1. 将插件放入 AstrBot 的插件目录
2. 重启 AstrBot
3. 在聊天中直接输入上述指令

## 备注

- 本插件以常见 TRPG 跑团骰点流程为核心，规则偏向简洁直观
